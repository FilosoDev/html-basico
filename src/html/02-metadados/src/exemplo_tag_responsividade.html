Exemplos de Tags de Responsividade

Essas tags são usadas para melhorar a experiência do usuário em dispositivos móveis, ajustando o layout e a aparência da página.

Exemplo 1: Meta Tag de Viewport

<meta name="viewport" content="width=device-width, initial-scale=1.0">

Essa tag ajusta a largura da página para o tamanho da tela do dispositivo e define o zoom inicial.

Exemplo 2: Meta Tag Mobile Web App Capable

<meta name="mobile-web-app-capable" content="yes">

Permite que a página seja exibida em tela cheia, como um aplicativo nativo em dispositivos móveis.

Exemplo 3: Meta Tag Apple Mobile Web App Status Bar Style

<meta name="apple-mobile-web-app-status-bar-style" content="black-translucent">

Personaliza a barra de status em dispositivos Apple, tornando-a preta e translúcida.

Exemplo 4: Meta Tag Theme Color

<meta name="theme-color" content="#ffffff">

Define a cor da barra de navegação para dispositivos móveis, neste caso, branca.

Exemplo 5: Meta Tag X-UA-Compatible

<meta http-equiv="X-UA-Compatible" content="IE=edge">

Garante que o Internet Explorer use a versão mais recente do motor de renderização.
