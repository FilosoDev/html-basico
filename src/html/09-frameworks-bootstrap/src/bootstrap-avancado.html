Accordion (Acordeão)

Organiza conteúdo de forma expansível

O sistema de grid do Bootstrap é baseado em Flexbox e permite criar layouts responsivos rapidamente. Ele consiste em:

  • Container: centraliza e limita a largura máxima
  • Row: agrupa colunas horizontalmente
  • Col: colunas flexíveis que se adaptam
Dica: Sempre use .container > .row > .col para estrutura correta

Classes utilitárias para espaçamento, cores, flexbox e muito mais:

Espaçamento:
  • m-* - margin
  • p-* - padding
  • mt-* - margin-top
  • mb-* - margin-bottom
Display:
  • d-flex - display flex
  • d-block - display block
  • d-none - display none
  • d-*-block - responsivo

Várias formas de customizar o Bootstrap:

  1. CSS Customizado: sobrescrever estilos específicos
  2. Variáveis Sass: modificar cores, tamanhos, breakpoints
  3. Bootstrap Build: incluir apenas componentes necessários
  4. Themes: usar themes prontos da comunidade
Exemplo de variável Sass:
$primary: #custom-color;
$border-radius: 0.75rem;
