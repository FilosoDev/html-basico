O que é CSS Interno?

CSS interno é uma maneira de adicionar estilos diretamente dentro do arquivo HTML. Ele é definido dentro da tag <style> no cabeçalho do documento.

Este método é útil para pequenas páginas ou quando você deseja aplicar estilos específicos a um único documento.
