CSS Syntax Explanation

What is CSS?

CSS stands for Cascading Style Sheets. It is used to style and layout web pages.

CSS Syntax

The syntax for a CSS rule consists of a selector and a declaration block.

Example: 
selector {
    property: value;
}

Selectors

A selector points to the HTML element you want to style.

This paragraph is styled with internal CSS.

Declarations

A declaration block contains one or more declarations separated by semicolons.

Example: 
selector {
    color: red;
    font-size: 20px;
}

Types of CSS

  • Inline CSS: Uses the style attribute within HTML elements.
  • Internal CSS: Uses a <style> element inside the <head> section.
  • External CSS: Uses an external CSS file linked via the <link> element.
