CSS Syntax Explanation

O que é CSS?

CSS significa Cascading Style Sheets. É usado para estilizar e fazer layout de páginas da web.

CSS Syntax

A sintaxe de uma regra CSS consiste em um seletor e um bloco de declaração.

Exenplo: 
                selector {
                    property: value;
                }

Seletores

Um seletor aponta para o elemento HTML que você deseja estilizar.

Este parágrafo é estilizado com CSS interno.

Declaração

Um bloco de declaração contém uma ou mais declarações separadas por ponto e vírgula.

Exemplo: 
                selector {
                    color: red;
                    font-size: 20px;
                }

Tipos de CSS

  • Inline CSS: Use o atributo style dentro da tag.
  • Internal CSS: Use a tage <style> dentro do elemento <head>
  • External CSS: Use um arquive externo referenciado via o elemento <link>

Esse parágrafo é estilizado por um arquivo externo
