1. DIV Básica com Flexbox

Uma div simples centralizada usando Flexbox com efeitos visuais:

✨ Clique em mim!
/* CSS para DIV centralizada com Flexbox */ .container { display: flex; /* Ativa Flexbox */ align-items: center; /* Centraliza verticalmente */ justify-content: center; /* Centraliza horizontalmente */ width: 100%; max-width: 400px; height: 200px; background: linear-gradient(45deg, #74b9ff, #0984e3); border: 3px solid #2d3436; border-radius: 12px; transition: all 0.3s ease; /* Animação suave */ } .container:hover { transform: scale(1.05) rotate(2deg); /* Aumenta e rotaciona */ } /* HTML */ <div class="container"> <div class="content">✨ Conteúdo</div> </div>

2. Layout com CSS Grid

CSS Grid para layouts responsivos bidimensionais:

🎯 Card 1

Este card faz parte de um grid responsivo que se adapta automaticamente ao tamanho da tela.

🚀 Card 2

O CSS Grid permite layouts complexos com poucas linhas de código, muito mais flexível que floats.

💡 Card 3

Cada card tem efeito hover que o eleva, criando uma experiência interativa moderna.

⚡ Card 4

O grid se reorganiza automaticamente em telas menores, mantendo a responsividade.

/* CSS Grid responsivo */ .grid-container { display: grid; grid-template-columns: repeat(auto-fit, minmax(250px, 1fr)); /* auto-fit: cria colunas automaticamente minmax(250px, 1fr): mínimo 250px, máximo 1 fração */ gap: 1.5rem; /* Espaçamento entre itens */ } .card { background: white; padding: 1.5rem; border-radius: 10px; transition: transform 0.3s ease; } .card:hover { transform: translateY(-8px); /* Move para cima no hover */ }

4. Tipos de Posicionamento CSS

Demonstração de position: relative, absolute e fixed:

Container com position: relative
O elemento laranja tem position: absolute e flutua com animação CSS.

ABS
/* Posicionamento absoluto */ .posicionamento-exemplo { position: relative; /* Container de referência */ height: 300px; } .elemento-absoluto { position: absolute; /* Posiciona relativo ao pai */ top: 20px; right: 20px; animation: float 3s ease-in-out infinite; } /* Animação de flutuação */ @keyframes float { 0%, 100% { transform: translateY(0); } 50% { transform: translateY(-20px); } }

📚 Propriedades CSS Essenciais para DIVs

/* DIMENSÕES */ width: 300px; /* Largura fixa */ width: 50%; /* Largura percentual */ width: 100vw; /* 100% da viewport */ max-width: 800px; /* Largura máxima */ min-width: 200px; /* Largura mínima */ height: 200px; /* Altura */ /* ESPAÇAMENTO */ margin: 20px; /* Margem externa */ margin: 10px 20px; /* Vertical | Horizontal */ margin: 5px 10px 15px 20px; /* Top | Right | Bottom | Left */ padding: 15px; /* Espaçamento interno */ /* BORDAS */ border: 2px solid #333; /* Largura | Estilo | Cor */ border-radius: 10px; /* Bordas arredondadas */ border-top: 1px solid red; /* Borda específica */ /* FUNDO */ background-color: #f0f0f0; /* Cor sólida */ background: linear-gradient(45deg, red, blue); /* Gradiente */ background-image: url('imagem.jpg'); /* Imagem */ /* LAYOUT - FLEXBOX */ display: flex; /* Ativa Flexbox */ flex-direction: row; /* Direção (row, column) */ justify-content: center; /* Alinhamento horizontal */ align-items: center; /* Alinhamento vertical */ flex-wrap: wrap; /* Permite quebra de linha */ gap: 20px; /* Espaçamento entre itens */ /* LAYOUT - CSS GRID */ display: grid; /* Ativa Grid */ grid-template-columns: 1fr 2fr 1fr; /* Colunas */ grid-template-rows: auto; /* Linhas */ grid-gap: 20px; /* Espaçamento */ /* POSICIONAMENTO */ position: static; /* Padrão - fluxo normal */ position: relative; /* Relativo à posição original */ position: absolute; /* Relativo ao pai positioned */ position: fixed; /* Fixo na viewport */ position: sticky; /* Combinação de relative e fixed */ /* COORDENADAS (para position: absolute, relative, fixed) */ top: 10px; /* Distância do topo */ right: 10px; /* Distância da direita */ bottom: 10px; /* Distância de baixo */ left: 10px; /* Distância da esquerda */ /* VISIBILIDADE */ display: block; /* Padrão das DIVs */ display: none; /* Oculta completamente */ display: inline; /* Comporta como texto */ display: inline-block; /* Híbrido */ visibility: hidden; /* Oculta mas ocupa espaço */ opacity: 0.5; /* Transparência (0-1) */ /* EFEITOS */ box-shadow: 0 4px 8px rgba(0,0,0,0.1); /* Sombra */ transform: rotate(45deg); /* Rotação */ transform: scale(1.1); /* Escala */ transform: translate(10px, 20px); /* Movimento */ transition: all 0.3s ease; /* Transição suave */ /* Z-INDEX (sobreposição) */ z-index: 1; /* Controla qual elemento fica na frente */ z-index: 999; /* Valores maiores ficam mais na frente */ /* OVERFLOW (conteúdo que transborda) */ overflow: visible; /* Padrão - mostra conteúdo que transborda */ overflow: hidden; /* Esconde conteúdo que transborda */ overflow: scroll; /* Sempre mostra barras de rolagem */ overflow: auto; /* Mostra rolagem apenas quando necessário */

🎯 Melhores Práticas

  • ✅ Use classes semânticas: .header, .sidebar, .content ao invés de .div1, .div2
  • ✅ Prefira Flexbox e Grid: São mais flexíveis que float e position
  • ✅ Mobile First: Design primeiro para mobile, depois desktop
  • ✅ Use rem/em: Unidades relativas são mais responsivas que px
  • ✅ Evite div-itis: Não use DIVs desnecessárias, prefira elementos semânticos
  • ✅ Organize CSS: Agrupe propriedades relacionadas (layout, aparência, animações)
  • ✅ Use comentários: Documente seções complexas do CSS
  • ✅ Teste responsividade: Verifique em diferentes tamanhos de tela
