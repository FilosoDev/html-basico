Importância do Contraste

É essencial garantir que o contraste entre o fundo e o texto seja suficiente para leitura confortável. O contraste recomendado pelo WCAG (Web Content Accessibility Guidelines) é de pelo menos 4.5:1 para texto normal e 3:1 para texto grande.
